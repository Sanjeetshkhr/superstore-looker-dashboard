# Run Guide

## What This Project Demonstrates
- Data preparation and transformation using Google Apps Script (JavaScript)
- Dashboard design and data visualization using Looker Studio
- Version control and project documentation using Git/GitHub

## What Was Coded
- Apps Script data preparation pipeline (DataPrep.js)
- Custom Google Sheets menu integration (Code.js)
- HTML Service web app with data refresh functionality (index.html)

## What Was Built in Looker Studio
- 3-page interactive dashboard with scorecards, charts, and cross-filters
- Calculated fields for Average Order Value, Profit Ratio, and Month
- Filtered scorecards and conditional formatting pivot table

## Where to Look First
1. Start with the live Looker Studio report (link in README)
2. Review the Apps Script code in the apps-script/ folder
3. Skim the documentation in docs/ and looker-studio/
