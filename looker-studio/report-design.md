# Report Design

## Page 1: Executive Overview
- Scorecards: Total Sales ($), Total Profit ($), Profit Ratio (%), Average Order Value ($)
- Time Series: Year-on-Year Sales by Month (2014–2017 comparison)
- Dual-Axis Bar Chart: Sales and Profit Ratio by Region
- Pie Chart: Segment by Profit Ratio
- Filters: Date Range, Region, Segment, Category

## Page 2: Discount & Loss Analysis
- Bar Chart: Profit Ratio by Discount Band
- Bubble Chart: Profit vs Order Count by Sub-Category (size = Profit Margin)
- Scorecards: Loss Orders (1,871), Loss Value ($-156K) — filtered to Loss Flag = Yes
- Pivot Table: Profit Ratio by Sub-Category × Region (conditional formatting: red/green)
- Stacked Bar: Loss Flag by Discount Band
- Filters: Date Range, Region, Segment, Category

## Page 3: Geography Intelligence
- Geo Map: Sales by State
- Table: Average Order Value by State (top 10)
- Dual-Axis Bar Chart: Profit Ratio and Order Count by Ship Mode
- Horizontal Bar Chart: Top 10 worst performing states by Profit Ratio
- Filters: Date Range, Region, Segment, Category

## Page 4: About This Dashboard
- Data source, tools used, metric definitions, author, GitHub link
